Напиши Python-скрипт, который:
1. Читает все доступные CSV и JSON файлы в директории /data.
    - CSV - парсит с пюмощью pandas или Csv.
    - JSON - парсит стандартной библиотекой json.
2. Читает все HTML-шаблоны из директории /templates.
3. При запуске выводит список:
    -доступных файлов с данными;
    -доступных шаблонов.
4. Дает пользователю выбрать (через input) один файл данных и один HTML-шаблон.
5. Выводит список доступных чеков (по invoice id) из выбранного файла.
6. Пользователь выбирает конкретный invoice id, и по нему генерируется PDF-документ:
    -данные подставляются в HTML-шаблон;
    -PDF сохраняется в директории /output.
7. После генерации автоматически открывает PDF в системной программе.
Требования:
-Используй библиотеку WeasyPrint для генерации PDF.
-Поддержка кириллицы (подключи шрифт DejaVu Sans или Roboto).
-Скрипт должен работать на Windows и macOS.
-Сделай аккуратный вывод меню в консоли (нумерация вариантов).
